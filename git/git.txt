1 查看远程分支
$ git branch -a

2 查看本地分支
$ git branch

3 创建分支
$ git branch test
$ git branch

4 切换分支到test
$ git branch
$ git checkout test

5 删除本地分支   
git branch -d xxxxx

7.提交pr

git submodule update --init

8.移除本地commit
git reset HEAD~  

9.存储修改
git stash
git stash list 查看列表
git stash clear 清空
